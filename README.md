# Jetbrains TeamCity environment

Deploys production-ready TeamCity to your machine using docker.

## Pre-requisites
* Docker
* `docker-compose`

## How to run
1. `$ cp .env.example .env`
2.  Modify environment variables at `.env` to feed your needs
3. `$ docker-compose up -d`
